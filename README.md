## 快速上手

<!-- add docs here for user -->

查看 [egg docs][egg] 详细文档.

### 开发

```
git clone http://git.sizhijie.com/szjcomo/egg-szjcomo.git egg-szjcomo-app
cd egg-szjcomo-app

extend  app.cache app.excel

```

```bash
$ npm i
$ npm run dev
$ open http://localhost:8105/
```

### 布署

```bash
$ npm start
$ npm stop
```
